Handy Codes

Getting a fresh local repo when it's compromised

  1. Backup local master - git checkout -b master-BACKUP
  2. Delete local master - git branch -D master
  3. Checkout remote master - git checkout --track origin/master

You forgot to set an upstream on remote for your branch.

  1. Set upstream for your branch on remote - git push --set-upstream origin [branch name]
