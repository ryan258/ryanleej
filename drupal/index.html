Drush Commands

  • drush pm-update
  • drush dl
  • drush en -y // will also dl/en dependencies, -y is auto yes to prompts
  • drush dis [project name] // disable modules
  • drush pm-uninstall [project name // does the same as the uninstall tab on the modules page, removes data from db]
  • drush pm -releasenotes // read release notes, especially before updating or
    drush rln [project name]
  • drush up -n | grep available // see what updates are available
  • drush pm-updatecode [project name] // update a project w/ details
    or drush upc [project name]
  • drush updatedb or: // update database
    drush updb
  • drush dl [full version ie module_filter-7.x-2.-dev] -y // Upgrade, downgrade to specific versions, then update db
  • drush cc all // clear cache
  • drush pm-update
    or drush up
  • drush cron // by default drupal runs drush every 3 hours
