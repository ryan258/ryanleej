Drush Commands

  • drush pm-update
  • drush dl
  • drush en -y // will also dl/en dependencies, -y is auto yes to prompts
  • drush dis [project name] // disable modules
  • drush pm-uninstall [project name // does the same as the uninstall tab on the modules page, removes data from db]
  • drush pm -releasenotes // read release notes, especially before updating or
    drush rln [project name]
  • drush up -n | grep available // see what updates are available
  • drush pm-updatecode [project name] // update a project w/ details
    or drush upc [project name]
  • drush updatedb or: // update database
    drush updb
  • drush dl [full version ie module_filter-7.x-2.-dev] -y // Upgrade, downgrade to specific versions, then update db
  • drush cc all // clear cache
  • drush pm-update
    or drush up
  • drush cron // by default drupal runs drush every 3 hours

Drupal Multilingual

With Drupal 7 and later You can:

  • http://example.com/admin/config/regional/language
  • Manage strings for localization: http://example.com/admin/config/regional/translate
  • Specify how the desired languages are detected: http://example.com/admin/config/regional/language/configure
  • or Configuration → Regional and language → Detection and selection
  • Translate strings: http://example.com/admin/config/regional/translate/translate
  • Import strings: http://example.com/admin/config/regional/translate/import
  • Export strings: http://example.com/admin/config/regional/translate/export

Setting up the Picture Module

Part 1 - Image Fields

Enable picture, breakpoint, and ctools modules Add breakpoints to your sub-theme's .info file to establish your break points in your theme. breakpoint[mobile] = (min-width: 0px) breakpoint[narrow] = (min-width: 740px) breakpoint[normal] = (min-width: 980px) breakpoint[wide] = (min-width: 1220px) if these don't show up in your theme go to the theme settings and scan for break points (which will be read from the .info file) config->breakpoints
